# 📚网站说明

本网站是知识库兼博客，内容为本人的学习经历所遇到的知识，没有具体的倾向。网站的搭建使用了vuepress-theme-vdoing (opens new window)主题，如果对本网站有任何问题可以联系我，联系方式在文章最后

# 使用指南

顶部导航栏：学习笔记、随笔、索引、友链、更多、Github

  • 学习笔记：学习过程中的各种笔记
  • 随笔：未整理归档的内容、个人日常、摘抄
  • 索引：
    • 分类：专栏分类
    • 标签：文章标签
    • 归档：按时间顺序将所有文章排序
  • 友链：友情链接
  • 更多：说明性文档或介绍性文档
  • Github： afroginawell的Github网站的链接

首页——实例&计算机理论：

  • 实例：学习过程中写的各种小玩意（源码在Github仓库）
  • 计算机理论：计算机组成原理、数据结构与算法等计算机理论知识

每篇文章后面自带评论功能，方便访客评论，禁止发布非法评论
