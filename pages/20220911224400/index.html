# 随笔2

蛮喜欢写博客的，9月初的时候用vuepress和github搭建了一个自己的博客。其实嘛，这个博客就是写给自己看的，把自己平时接触到的各种知识趣事统合到博客上，方便自己随时查看。

平时看书的时候就喜欢在书上写写画画，把不懂得喜欢的重要的都标出来。但时间长了，往往就会忘记好多，再去翻看也是匆匆翻过，没有了第一次翻看的那种感觉，就想着将这些标出来的都记录下来，再整理成体系，当自己感觉哪部分非常吸引自己或者忘记了，可以根据自己的记录再去翻看，这种带有一定目的性的翻看就会显得很专注，特别舒服。不过这是在看名著、小说之类的书。

而对于专业书，往往要求会更加高，就会准备一个笔记本进行更加详细的记录，毕竟专业书一般空白地方比较少，全记在书上，就会显得很乱。那种一句话好多术语的最讨厌了，弄懂这些术语都要写很多，而每个术语都又会引出其他的术语，显得特别麻烦。不过我觉得这就是一种学习，将知识编制成一张网，不断地拓宽。如果一部分内不懂的特别多，还可以专门搞个专题，方便自己记忆。

当然这些最后都会汇聚到博客上，随时随地可以进行翻看，偶尔就会觉得蛮有成就感的。自己搭博客还有评论功能，当对文章有啥疑问，不方便及时修改，还可以用评论来记录一下，就觉得非常棒。

博客的地址这就不写了，要不博客被恶意访问我得哭死。
